# Política de Seguridad

## Versiones Soportadas

Actualmente mantenemos soporte de seguridad para las siguientes versiones de Radio Satelital:

| Versión | Soportada          |
| ------- | ------------------ |
| 9.5.x   | :white_check_mark: |
| 8.5.x   | :white_check_mark: |
| < 8.0   | :x:                |

## Reportar una Vulnerabilidad

Si descubre una vulnerabilidad de seguridad en este proyecto, por favor NO abra un "Issue" público.

En su lugar, por favor contacte directamente al desarrollador principal (Latán Villegas) a través de las redes integradas en la aplicación o mediante un mensaje privado en GitHub.

Nos comprometemos a investigar todas las vulnerabilidades legítimas y solucionarlas lo antes posible.
