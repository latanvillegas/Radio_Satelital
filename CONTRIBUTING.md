# Contribuir a Radio Satelital

¡Gracias por tu interés en contribuir a Radio Satelital!

## Cómo contribuir
1. **Reportar Bugs:** Usa la pestaña de "Issues" en GitHub si encuentras un error en la reproducción o la interfaz.
2. **Sugerir Emisoras:** Puedes abrir un Issue con el título "Solicitud de Emisora: [Nombre]" incluyendo la URL del stream (HTTPS).

## Estilo de Código
* Mantenemos el código JS limpio y sin dependencias externas pesadas (Vanilla JS).
* Respetamos la estructura de variables CSS definida en `style.css`.

## Pull Requests
1. Haz un Fork del repositorio.
2. Crea una rama para tu mejora (`git checkout -b mejora/nueva-funcion`).
3. Envía tu Pull Request describiendo claramente los cambios.
